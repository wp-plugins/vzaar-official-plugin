== Description ==

vzaar official media manager makes it easy to upload, manage, and embed your vzaar videos directly from your WordPress Installation

Please note: This requires use of our API, available in the Pro Toolkit. For more information, please enquire with sales@vzaar.com

== Installation ==

1. Activate the plugin through the 'Plugins' menu in WordPress

2. Click the vzaar tab and enter your API token and username

== Frequently asked questions ==

= Where can I get an API token =

You can find this in your vzaar settings. This requires an account with our Pro Toolkit. To upgrade, please contact sales@vzaar.com
